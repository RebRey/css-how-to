Making a Navigation Bar by Using Bootstrap


Add Bootstrap to Your Project

  1. Copy-paste the stylesheet link into your head before all other stylesheets to load our CSS.
  2. Copy-paste the Popper script and JS script to use their functionallities.
  3. If you decide to go with the separate scripts solution, Popper must come first (if you're using tooltips or popovers), and then our JavaScript plugins.

Creating the Nav Bar

  • Use the classes that are provided by Bootstrap.
  • Navbars require a wrapping .navbar with .navbar-expand{-sm|-md|-lg|-xl|-xxl} for responsive collapsing and color scheme classes.
  • Navbars and their contents are fluid by default. Change the container to limit their horizontal width in different ways.
  • Use our spacing and flex utility classes for controlling spacing and alignment within navbars.
  • Navbars are responsive by default, but you can easily modify them to change that. Responsive behavior depends on our Collapse JavaScript plugin.
  • Ensure accessibility by using a nav element or, if using a more generic element such as a div, add a role="navigation" to every navbar to explicitly identify it as a landmark region for users of assistive technologies.
  • Indicate the current item by using aria-current="page" for the current page or aria-current="true" for the current item in a set.

Adding a Background Color:

  • This example uses style (navbar-dark) and background (bg-dark).
  • You can also use style (navbar-light) and background (bg-light).
  • You can visit the Buttons documentation to view the color classes and change the nav bar background by changing the bg class section.

Brand

  • The .navbar-brand can be applied to most elements, but an anchor works best, as some elements might require utility classes or custom styles.

Customizing

  • Identify the container that holds the nav bar. In this case it is an ul (unordered list).
  • The unordered list was moved the right by adding another class that adds a margin at the starf of the unordered list so that it will push the group to the right.
  • Go to the Utilities documentation and click on Spacing and view the margin and padding section
  • In this example we choose to add a margin (m) to the start (s) of the class and add a bit of margin (auto) to the left of the ul.
  • The class used was ms-auto

Responsive Behaviors

  • Navbars can use .navbar-toggler, .navbar-collapse, and .navbar-expand{-sm|-md|-lg|-xl|-xxl} classes to determine when their content collapses behind a button. In combination with other utilities, you can easily choose when to show or hide particular elements.
